# Ordo

<img src="public/logo.svg" alt="Ordo" width="64" height="64" />

A minimal kanban board with an extremely simple feature set.

## Features

- Create boards with multiple columns (e.g. To Do, In Progress, Done)
- Add tickets to columns with just a title and description
- Drag and drop tickets between columns
- Simple, clean UI

## Tech Stack

- [Convex](https://convex.dev/) - Backend (database, real-time sync)
- [React](https://react.dev/) - Frontend
- [Vite](https://vitest.dev/) - Build tool
- [Tailwind](https://tailwindcss.com/) - Styling

## Get started

```
npm install
npm run dev
```
